Setup Guide
  1. Create a Facebook App at developers.facebook.com
  2. Add the Instagram Graph API product
  3. Link your Instagram Business/Creator account to a Facebook Page
  4. Generate a User Access Token with required permissions
  5. Exchange the short-lived token for a long-lived token
  6. Enter the token and your Instagram User ID above
