## instagram_manage
Manage Instagram media: delete posts.

**Note:** Only media published via the Content Publishing API can be deleted. Organic posts cannot be deleted via the API.

**Arguments:**
- **action** (string, required): "delete"
- **media_id** (string, required): Media ID of the post to delete

~~~json
{"action": "delete", "media_id": "17895695668004550"}
~~~
